# WhatsApp Setup Guide

## Hardcoded WhatsApp Number Setup
1. Open the source code of the project in your preferred IDE.
2. Navigate to the configuration file where the WhatsApp number is defined.
3. Replace the placeholder number with your hardcoded WhatsApp number.
4. Save the changes.

## Device Linking Process
1. Ensure your smartphone has the WhatsApp application installed.
2. Open the application and navigate to the settings.
3. Choose the option to link a new device.
4. Scan the QR code displayed on your source code setup page.
5. Confirm the successful linking of your device.
6. Test the integration by sending a message to verify that everything is working correctly.
